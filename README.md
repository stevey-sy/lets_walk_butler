# lets_walk_butler
반려견의 산책 &amp; 일상 기록이 가능한 App <집사야, 산책 가즈아!> 의 소스코드 입니다.

# 구현한 기능

- 오늘의 날씨
- 반려견 프로필, 식사일지, 산책일지 기록
- 산책량 측정 기능
- 산책 알람 기능

# 활용한 라이브러리

- Retrofit
- Glide
- TedPermission
- 

# 활용한 API

- Google Map
- Open Weather Map
